Timestamp Microservice

What does this do?

Pass a string as a parameter, and it will check to see whether that string is a unix timestamp or a natural language date. If it is a valid form of either one it will return both the unix and natural language format of the date as a JSON response. Invalid dates return in the same format but with null as the value for both date types.

To make things simple I've provided a few strings to get started with. You can click on them to see it work, or enter your own data into the querystring.
